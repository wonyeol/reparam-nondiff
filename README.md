## Code
You can perform the experiments shown in the above NIPS'18 paper, by running the following script.
```
./script.sh 0.001 1 16
./script.sh 0.001 8  0
./script.sh 0.001 16 0
./script.sh 0.01  1 16
./script.sh 0.01  8  0
./script.sh 0.01  16 0
```
